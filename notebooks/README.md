# Notebooks

To get started, go to `notebooks` (if you are not already there).
Then execute


```bash
python -m venv venv
source ven/bin/activate
```
(on UNIX-like systems)

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```
(on Windows / PowerShell)

```bash
pip install -r requirements.txt
jupyter-lab
```
